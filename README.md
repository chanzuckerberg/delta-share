## Delta Share Token Service
Databricks Delta Share allows us to share data within Databricks delta tables to external partners without having to copy or replicate the data. This tool helps streamline the process of rotating usersâ€™ tokens provided by the Databricks API to access the shared data.
This project is under active development.

### How to install
1. `brew tap chanzuckerberg/tap`
2. `brew install chanzuckerberg/tap/delta-share`

### How to run
1. `delta-share`
